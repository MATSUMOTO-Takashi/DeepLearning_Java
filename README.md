## 実装する上で注意しておく点（戒め）
- 著者が配布しているサンプルコードも参照しつつ、基本的にはコピペをせずに自分で実装していくようにする
- 書籍にもコード上にも説明のない部分があったりするので、その辺は都度調べながらコメントを書き足していく
